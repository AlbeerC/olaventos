🗺️ Configuración de Google Maps API para OLAEVENTOS

Para que el mapa funcione correctamente, necesitas configurar Google Maps API. Sigue estos pasos:

📝 Paso 1: Crear un proyecto en Google Cloud Console

  1. Ve a Google Cloud Console
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Habilita la API de Google Maps JavaScript

🔑 Paso 2: Obtener tu API Key

  1. Ve a "Credenciales" en el menú lateral
  2. Haz clic en "Crear credenciales" → "Clave de API"
  3. Copia tu clave de API
  4. Configura las restricciones de dominio para mayor seguridad

⚙️ Paso 3: Configurar la API Key en el código

Reemplaza YOUR_API_KEY en el archivo index.html:

<script async defer src="https://maps.googleapis.com/maps/api/js?key=TU_API_KEY_AQUI&callback=initMap&libraries=geometry"></script>

🌍 Paso 4: APIs necesarias

Asegúrate de habilitar estas APIs en Google Cloud Console:

  • ✅ Maps JavaScript API
  • ✅ Geocoding API (opcional, para búsquedas)
  • ✅ Places API (opcional, para autocompletado)

⚠️ Importante

  • Nunca expongas tu API Key en repositorios públicos
  • Configura restricciones de dominio en Google Cloud Console
  • Monitorea el uso y facturación regularmente
  • Google Maps ofrece $200 USD gratis por mes
